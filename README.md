# 피로도 (Pirodo)

일상의 다양한 활동을 추적하고 현재 피로도를 측정해주는 모바일 앱입니다.

## 주요 기능

### 피로도 측정
- 업무, 수면, 운동, 스크린 타임 등 다양한 활동 추적
- 과학적 가중치 기반 피로도 계산 (0-100%)
- 4단계 피로도 레벨 (완전 충전, 양호, 피곤함, 탈진)

### 재미있는 피드백
- 피로도에 따른 유머러스한 메시지
- 이모지와 컬러로 직관적인 상태 표시
- 개인화된 추천 사항 제공

### 활동 관리
- 11가지 활동 타입 지원
  - **피로 증가**: 업무, 스크린 타임, 앉아있기, 스트레스, 카페인
  - **회복**: 수면, 운동, 휴식, 실외 활동, 물 섭취, 식사
- 활동별 시간 및 메모 기록
- 상세한 활동 히스토리

### 데이터 저장
- 로컬 저장소에 데이터 자동 저장
- 날짜별 데이터 관리 (자정에 자동 초기화)

## 기술 스택

- **React Native 0.73.2** - 크로스 플랫폼 모바일 앱
- **TypeScript** - 타입 안정성
- **React Navigation** - 화면 네비게이션
- **AsyncStorage** - 로컬 데이터 저장
- **Context API** - 상태 관리

## 설치 및 실행

### 요구사항
- Node.js >= 18
- npm 또는 yarn
- React Native 개발 환경 설정
  - Android: Android Studio, JDK
  - iOS: Xcode, CocoaPods (macOS만 해당)

### 설치
```bash
# 의존성 설치
npm install

# iOS의 경우 (macOS만)
cd ios && pod install && cd ..
```

### 실행
```bash
# Android
npm run android

# iOS (macOS만)
npm run ios

# Metro 번들러만 시작
npm start
```

## 피로도 계산 로직

### 기본 계산식
```
피로도 = 기본 피로도(50) + Σ(활동 영향) + 보너스/페널티
```

### 활동 영향
각 활동은 시간과 가중치를 곱해서 계산:
```
활동 영향 = (시간(분) / 60) × 가중치 × 100
```

### 가중치 예시
- **수면**: -0.35 (회복)
- **스트레스**: +0.25 (피로)
- **업무**: +0.20 (피로)
- **휴식**: -0.20 (회복)
- **운동**: -0.15 (회복)

### 특별 보너스/페널티
- 수면 부족(< 5시간): +15% 페널티
- 적정 수면(7-9시간): -10% 보너스
- 업무 과다(> 10시간) + 휴식 부족(< 1시간): +10% 페널티

## 화면 구성

### 1. 홈 화면
- 현재 피로도를 큰 원형 게이지로 표시
- 재미있는 메시지와 추천 사항
- 활동 추가 및 상세 보기 버튼

### 2. 활동 추가 화면
- 11가지 활동 타입 선택
- 시간 입력 (시/분)
- 메모 추가 (선택사항)

### 3. 상세 분석 화면
- 활동 통계 요약
- 활동 타입별 그룹화
- 개별 활동 목록 및 삭제 기능
